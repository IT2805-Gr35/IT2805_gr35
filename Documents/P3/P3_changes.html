The changes we have made

Changes to index.html: Logo and icons got a bit smaller for looks. Search bar was added to the hambur menu for prettier spacing.

Changes to about.html: For practical reasons we decided to devide the about-infromation into three sections. We also did this to make it more intuitive for the reader. The sections move upwards when you hover over it. I also decided to not have any pictures in the about section

Changes to contact.html: Changed the field structure of the original design, in order to remove undesired user inputs like zip or adress. Decided to make background of input fields white to improve readability against the background. Changed the layout to a centered single column with input label above the input field for improved readability and ease of universial use across platforms. Made submit button of same width as the rows above, color green with white text in order to increase visibility for user. Button changes to darker colour on hover.

Changes to fruit.html, berries.html, mushrooms.html:Changed layout from items in double column to single column layout. Also changed the content of each item to just contain the item name and a short description.

Changes to information.html: There have been some changes in the design of the page. Initially, there was not supposed to be a caption between the headline and the slideshow, but this was added as it looked a bit better. I also separated the info box into two parts, one with language and one with general info.

Additional Javascript file clickDirector.js: Javascript needed to find the clicked item and give that information to pushInfo.js

Additional Javascript file hamburger.js: Javascript needed to make the functionality of the hamburger menu

Additional Javascript file headerFooter.js: Script needed for mudularity and pushing header and footer to each page

Additional Javascript file itemArray.js: Script needed to store the information we need for each list-item as well as information to be loaded in info.html

Additional Javascript file pullItems.js: Script needed to create each list item using the information stored in itemArray.js

Additional Javascript file pushInfo.js: Script needed to push content to info.html so that we can have just one html file for it

Javascript file validate_form.js: Formerly cleanup.js, just renamed
