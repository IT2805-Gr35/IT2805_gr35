The changes we have made

Changes to index.html:

Changes to about.html:For practical reasons we decided to devide the about-infromation into three sections. We also did this to make it more intuitive for the reader. The sections move upwards when you hover over it. I also decided to not have any pictures in the about section

Changes to contact.html:Changed the field structure of the original design, in order to remove undesired user inputs like zip or adress. Decided to make background of input fields white to improve readability against the background. Changed the layout to a centered single column with input label above the input field for improved readability and ease of universial use across platforms. Made submit button of same width as the rows above, color green with white text in order to increase visibility for user. Button changes to darker colour on hover.
